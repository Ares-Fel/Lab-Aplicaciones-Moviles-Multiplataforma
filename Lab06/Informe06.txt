Conclusiones

* Mediante un navigation podemos dirigirnos a otra vista si le asignamos una funcionalidad como un boton, al darle clic la vista asignada se abrira.
* Podemos crear las vistas en una carpeta distinta para mantener u orden, pero lueg deberan ser llamadas en app.js para ser usadas, las acciones seran seran las mismas.
* El paso de variables entre vistas es posible, mediante una variable que lo alamcena , la otra vista recibe esa variable para ser utilizada.
* Depeniendo si es createrDrawerNavigater o createBottomNavigator podemos asignarle el comportamiento entre vistas, es decir si es mediante panel lateral o botones de navegacion.
